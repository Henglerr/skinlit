# SkinScore Legal Pages (GitHub Pages)

This folder contains the public legal pages used by the iOS app:

- `privacy.html`
- `terms.html`
- `support.html`

## Publish steps

1. Push this repository to `main` (or your default branch).
2. In GitHub, open repository settings:
   - `Settings` -> `Pages`
3. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
4. Save and wait a few minutes for deployment.

Expected base URL:

`https://henglerr.github.io/Skinapp/`

Final links:

- `https://henglerr.github.io/Skinapp/privacy.html`
- `https://henglerr.github.io/Skinapp/terms.html`
- `https://henglerr.github.io/Skinapp/support.html`
