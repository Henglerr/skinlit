Place the 25 calibrated skin reference images in this folder.

- ref-001 ... ref-005 for range 0-2
- ref-006 ... ref-010 for range 2-4
- ref-011 ... ref-015 for range 4-6
- ref-016 ... ref-020 for range 6-8
- ref-021 ... ref-025 for range 8-10

Accepted extensions:
- .jpg
- .jpeg
- .png
- .webp
- .heic
